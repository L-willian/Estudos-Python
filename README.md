# Estudos de Python 🐍

Repositório de estudos e projetos práticos em Python, organizado em desafios sequenciais e pequenos projetos integradores. O objetivo é consolidar conhecimentos fundamentais da linguagem por meio de exercícios aplicados.

## Visão Geral

- Desafios: exercícios focados em conceitos básicos e estruturais (variáveis, controle de fluxo, estruturas de dados, funções, exceções).
- Mini-projetos integradores: aplicações pequenas que combinam vários conceitos para resolver problemas práticos.

## Estrutura do Repositório

- `/Desafios/` — Conjunto de desafios numerados, com explicações e exemplos.
- `/Mini-Projetos Integradores/` — Projetos compactos que aplicam os conceitos aprendidos.

## Como Contribuir

1. Faça um fork do repositório.
2. Crie uma branch com sua feature: `git checkout -b feat/minha-melhoria`.
3. Faça commits claros e atômicos.
4. Abra um Pull Request descrevendo as mudanças.

## Como Executar

Pré-requisitos:
- Python 3.7+

Executar um arquivo de exemplo:

```bash
cd Estudos-Python
python "Desafios/Desafio1.py"
python "Mini-Projetos Integradores/Sistema de Cadastro de Produtos.py"
```

## Boas práticas

- Use nomes de variáveis descritivos
- Escreva funções curtas e com responsabilidade única
- Adicione docstrings e comentários quando necessário

## Contato

Para dúvidas ou sugestões, abra uma issue no repositório.
