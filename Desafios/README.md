# Estudos de Python 🐍

Um repositório dedicado ao aprendizado e prática de conceitos fundamentais da linguagem Python através de desafios práticos e progressivos.

## 📋 Descrição

Este projeto contém uma série de desafios progressivos em Python, desde conceitos básicos como variáveis e tipagem dinâmica até operações mais complexas com estruturas de dados. Ideal para quem está iniciando seus estudos em Python ou deseja revisar e praticar conceitos essenciais.

## 🎯 Objetivos

- Compreender os conceitos fundamentais de Python
- Praticar lógica de programação através de desafios
- Solidificar conhecimento em tipos de dados, estruturas de controle e manipulação de dados
- Construir uma base sólida para programação orientada a objetos e funcional

## 🚀 Como Executar

### Pré-requisitos

- Python 3.7 ou superior instalado
- Git (para clonar o repositório)

### Instalação

```bash
# Clonar o repositório
git clone https://github.com/L-willian/Estudos-Python.git

# Navegar até o diretório
cd Estudos-Python
```

### Executando os Desafios

Para executar um desafio específico:

```bash
python Desafio1.py
python Desafio2.py
# ... e assim por diante
```
