# Desafios — Estudos de Python

Esta pasta reúne exercícios progressivos destinados a reforçar fundamentos de Python. Cada arquivo `DesafioN.py` foca em um conceito específico, com exemplos e saídas demonstrativas.

## Lista de Desafios

| Desafio | Arquivo | Objetivo |
|--------:|---------|----------|
| 1 | `Desafio1.py` | Variáveis e tipagem dinâmica — cálculo de área de um triângulo e uso de tipos numéricos |
| 2 | `Desafio2.py` | Estruturas de controle — condicionais `if/elif/else` e operador ternário |
| 3 | `Desafio3.py` | Laços de repetição — `for` e lógica para identificar pares/ímpares e acumular somas |
| 4 | `Desafio4.py` | Listas — operações básicas (`append`, `pop`), iteração e indexação |
| 5 | `Desafio5.py` | Funções — definição, retorno de valores e composição de funções simples |
| 6 | `Desafio6.py` | Dicionários — criação, leitura, atualização e formatação de saída |
| 7 | `Desafio7.py` | Tratamento de exceções — `try/except` para divisões e acesso seguro a chaves de dicionários |

## Como executar

Navegue até a pasta do repositório e execute o script desejado com Python:

```bash
python "Desafios/Desafio1.py"
```

## Recomendações de estudo

- Leia o código antes de executar para entender a intenção do exercício.
- Faça alterações: recalcule valores, adicione tratamento de entrada do usuário e escreva testes simples.
- Documente melhorias no próprio arquivo com comentários e docstrings.

## Sugestões de extensão

- Adicionar entrada interativa via `input()` para permitir testes com diferentes valores.
- Encapsular lógica em funções para facilitar reuso e testes.
- Criar testes simples com `unittest` ou `pytest` para validar saídas esperadas.
